HOW TO UPDATE YOUR .env FILE
================================

Current .env file location:
C:\Users\sivak\OneDrive\Documents\textile\.env

STEP 1: Open the .env file with Notepad
----------------------------------------
Right-click on .env → Open with → Notepad

STEP 2: Replace the GMAIL_APP_PASSWORD line
--------------------------------------------
CHANGE FROM:
GMAIL_APP_PASSWORD=sivakathir0911@

CHANGE TO:
GMAIL_APP_PASSWORD=your_16_char_password_here

Example (use YOUR generated password):
GMAIL_APP_PASSWORD=abcdefghijklmnop

IMPORTANT:
- Use the 16-character App Password from Google
- NO SPACES in the password
- NO QUOTES around the password
- Just the password directly

STEP 3: Save the file
---------------------
File → Save

STEP 4: Restart Flask
---------------------
In terminal:
- Press Ctrl+C to stop Flask
- Run: python app.py

STEP 5: Test
------------
- Register with any email
- OTP should be sent to that email!

================================

CORRECT FORMAT EXAMPLE:

SECRET_KEY=your-secret-key-2002
GMAIL_ADDRESS=sivakathir785@gmail.com
GMAIL_APP_PASSWORD=abcdefghijklmnop

(Replace abcdefghijklmnop with YOUR 16-char App Password)
